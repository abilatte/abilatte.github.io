## 目录与路径

### cd (change directory)

>   cd - 切换当前目录



| 写法  | 作用                 |
| ----- | -------------------- |
| cd .. | 当前目录的上一级目录 |
| cd .  | 当前目录             |
| cd ~  | 家目录               |
| cd -  | 上一次访问的目录     |



使用方式: 

```shell
abilatte@ubuntu:~/code$ pwd
/home/abilatte/code
abilatte@ubuntu:~/code$ cd a/
abilatte@ubuntu:~/code/a$ cd ..
abilatte@ubuntu:~/code$ cd -
/home/abilatte/code/a
abilatte@ubuntu:~/code/a$ cd ~
abilatte@ubuntu:~$
```



## 查看目录

###  ls (list)

>   ls - 列出目录内容, 默认忽略以`.`开头的文件或目录



| 参数 | 作用                                 |
| ---- | ------------------------------------ |
| -a   | 显示所有文件和目录, 不忽略隐藏的内容 |
| -l   | 显示文件和目录的详细信息             |



```shell
abilatte@ubuntu:~$ ls
code  snap  下载  公共  图片  文档  桌面  模板  视频  音乐
```

`-a` : 显示所有的文件和目录

```shell
abilatte@ubuntu:~$ ls -a
.  ..  .bash_history  .bash_logout  .bashrc  .cache  .config  .dotnet  .local  .profile  .ssh  .vscode-server  .wget-hsts  code  snap  下载  公共  图片  文档  桌面  模板  视频  音乐
```

`-l` : 显示文件和目录的详细信息, 其中包括: **文件类型, 文件所有者权限, 文件所属组权限, 其他用户权限, 硬链接数, 文件所有者, 文件所属组, 文件大小, 修改日期, 文件名** 

```shell
abilatte@ubuntu:~$ ls -l
total 40
drwxrwxr-x 2 abilatte abilatte 4096 Jun 27 05:29 code
drwx------ 6 abilatte abilatte 4096 Jun 26 23:53 snap
...
```

```shell

-rw-r--r--  1 abilatte abilatte  807 Feb 13 20:16 .profile
```

>   查看文件详细信息的时候, 还有一种写法 `ll `: 
>
>   有的版本中 `ll` 等价于 `ls -l`
>
>   有的版本中 `ll` 等价于 `ls -laF`
>
>   可以使用`alias ll`查看



## 创建与删除

### mkdir (make directory)

>   mkdir - 如果指定的目录不存在, 则创建



```shell
abilatte@ubuntu:~/code$ ll
total 8
drwxrwxr-x  2 abilatte abilatte 4096 Jul 12 08:14 ./
drwxr-x--- 18 abilatte abilatte 4096 Jun 27 05:29 ../

abilatte@ubuntu:~/code$ mkdir a
abilatte@ubuntu:~/code$ ll
total 12
drwxrwxr-x  3 abilatte abilatte 4096 Jul 12 08:14 ./
drwxr-x--- 18 abilatte abilatte 4096 Jun 27 05:29 ../
drwxrwxr-x  2 abilatte abilatte 4096 Jul 12 08:14 a/

abilatte@ubuntu:~/code$ mkdir a
mkdir: a: File exists
```



如果要创建多层目录, 需要使用参数`-p`

```shell
abilatte@ubuntu:~/code$ mkdir -p a/b/c
abilatte@ubuntu:~/code$ tree
.
└── a
    └── b
        └── c
```



### rmdir (remove directory)

>   rmdir - 如果指定目录为空, 则删除



```shell
abilatte@ubuntu:~/code$ tree
.
└── a
    └── test

abilatte@ubuntu:~/code$ rmdir a
rmdir: failed to remove 'a': Directory not empty

abilatte@ubuntu:~/code$ rm a/test 
abilatte@ubuntu:~/code$ tree
.
└── a

abilatte@ubuntu:~/code$ rmdir a
abilatte@ubuntu:~/code$ ll
total 8
drwxrwxr-x  2 abilatte abilatte 4096 Jul 12 08:16 ./
drwxr-x--- 18 abilatte abilatte 4096 Jun 27 05:29 ../

```



### rm (remove)

>   rm - 删除文件或目录
>



| 参数 | 作用                                 |
| ---- | ------------------------------------ |
| -r   | 递归删除目录以及其中的内容           |
| -f   | 忽略不存在的文件，并且不进行交互确认 |
| -i   | 每次删除前提示                       |



`-r` : 递归删除目录以及其中的内容


```shell
abilatte@ubuntu:~/code$ tree
.
└── a

abilatte@ubuntu:~/code$ rm a
rm: 无法删除 'a': 是一个目录
abilatte@ubuntu:~/code$ rm -r a

abilatte@ubuntu:~/code$ ll
total 8
drwxrwxr-x  2 abilatte abilatte 4096 Jul 12 08:27 ./
drwxr-x--- 18 abilatte abilatte 4096 Jun 27 05:29 ../
```

`-i` : 删除前提示

```shell
abilatte@ubuntu:~/code$ ll
total 8
drwxrwxr-x  2 abilatte abilatte 4096 Jul 12 08:29 ./
drwxr-x--- 18 abilatte abilatte 4096 Jun 27 05:29 ../
-rw-rw-r--  1 abilatte abilatte    0 Jul 12 08:29 test.txt

abilatte@ubuntu:~/code$ rm -i test.txt 
rm：是否删除regular empty file 'test.txt'？ y

abilatte@ubuntu:~/code$ ll
total 8
drwxrwxr-x  2 abilatte abilatte 4096 Jul 12 08:29 ./
drwxr-x--- 18 abilatte abilatte 4096 Jun 27 05:29 ../
```

`-f` : 忽略不存在的文件

```shell
# 文件存在时，二者效果相同
abilatte@ubuntu:~/code$ rm test.txt
abilatte@ubuntu:~/code$ rm -f test.txt

# 文件不存在时，rm 会报错并返回失败
abilatte@ubuntu:~/code$ rm test.txt
rm: 无法删除 'test.txt': 没有那个文件或目录
abilatte@ubuntu:~/code$ echo $?
1

# rm -f 会忽略不存在的文件并返回成功
abilatte@ubuntu:~/code$ rm -f test.txt
abilatte@ubuntu:~/code$ echo $?
0
```



## 复制与移动

### cp (copy)

>   cp - 拷贝文件或者目录

拷贝文件 : 文件不存在得到新文件, 文件存在就覆盖

```shell
abilatte@ubuntu:~/code$ touch test1.txt
# 如果目标文件已存在, 则会覆盖目标文件
abilatte@ubuntu:~/code$ cp test1.txt test2.txt 
abilatte@ubuntu:~/code$ ll
total 8
drwxrwxr-x  2 abilatte abilatte 4096 Jul 12 08:56 ./
drwxr-x--- 18 abilatte abilatte 4096 Jun 27 05:29 ../
-rw-rw-r--  1 abilatte abilatte    0 Jul 12 08:56 test1.txt
-rw-rw-r--  1 abilatte abilatte    0 Jul 12 08:56 test2.txt
```

拷贝目录 : 需要使用参数`-r`, 目录不存在得到新目录, 目录存在则源目录被拷贝到存在的目录中

```shell
abilatte@ubuntu:~/code$ mkdir a
# 如果目标目录已存在, 则A会被拷贝到B中
abilatte@ubuntu:~/code$ cp -r a b
abilatte@ubuntu:~/code$ ll
total 16
drwxrwxr-x  4 abilatte abilatte 4096 Jul 12 08:59 ./
drwxr-x--- 18 abilatte abilatte 4096 Jun 27 05:29 ../
drwxrwxr-x  2 abilatte abilatte 4096 Jul 12 08:58 a/
drwxrwxr-x  2 abilatte abilatte 4096 Jul 12 08:59 b/
```



### mv (move)

>   mv - 移动或重命名文件和目录

移动 / 重命名 : 	根据目标路径是否为已存在目录, 决定执行移动还是重命名, 目标文件已存在时可能被覆盖

```shell
abilatte@ubuntu:~/code$ touch test.txt
abilatte@ubuntu:~/code$ mkdir a
abilatte@ubuntu:~/code$ mv test.txt a
abilatte@ubuntu:~/code$ tree
.
└── a
    └── test.txt
```

```shell
abilatte@ubuntu:~/code$ mv test.txt test2.txt
abilatte@ubuntu:~/code$ ll
total 8
drwxrwxr-x  2 abilatte abilatte 4096 Jul 12 09:03 ./
drwxr-x--- 18 abilatte abilatte 4096 Jun 27 05:29 ../
-rw-rw-r--  1 abilatte abilatte    0 Jul 12 09:03 test2.txt
```

覆盖 : 源文件的内容覆盖目标文件, 源文件删除

```shell
abilatte@ubuntu:~/code$ touch test1.txt test2.txt
abilatte@ubuntu:~/code$ mv test1.txt test2.txt 
abilatte@ubuntu:~/code$ ll
total 8
drwxrwxr-x  2 abilatte abilatte 4096 Jul 12 09:05 ./
drwxr-x--- 18 abilatte abilatte 4096 Jun 27 05:29 ../
-rw-rw-r--  1 abilatte abilatte    0 Jul 12 09:05 test2.txt
```



## 查看文件内容

>   常用的方式主要是`vim`, 下面说一些其他的方式

### cat

>   适合直接查看较短文件, 文件较长时, 内容会快速滚过屏幕

```shell
cat filename
```

### more / less

>   这两个指令差不多, 都可以以翻页或者滚动的方式查看文件内容

```shell
more filename
less filename
```

### head

>   查看文件头部若干行的内容

```shell
# 默认显示十行
head filename
# 显示指定x行内容
head -n x filename
```

### tail

>   查看文件尾部若干行的内容

```shell
# 默认显示十行
tail filename
# 显示指定x行内容
tail -n x filename
```



## 文件属性

### chmod (change mode)

>   chmod - 修改文件和目录的权限
>

数字设定法 : 

```shell
# chmod xxx filename
# x : 8进制表示的权限
# 三位分别是文件所有者, 文件所属组, 其他用户
# 以二进制的三位分别表示
# 读, 写, 执行
# 101 -> 5

abilatte@ubuntu:~/code$ touch test
abilatte@ubuntu:~/code$ ll test
-rw-rw-r--  1 abilatte abilatte    0 Jul 12 09:42 test

abilatte@ubuntu:~/code$ chmod 775 test 
abilatte@ubuntu:~/code$ ll test
-rwxrwxr-x  1 abilatte abilatte    0 Jul 12 09:42 test*
```

文字设定法

```shell
# chmod [ugoa][+-=][rwx-] filename
# ugoa - user : 文件所有者
#	   - group : 文件所属组
#      - other : 其他用户
#      - all : 所有用户
# +-=  - + : 添加权限
#      - - : 移除权限
#      - = : 设定权限
# rwx- - r : 读权限
#	   - w : 写权限
#      - x : 执行权限
#      - - : 无权限

abilatte@ubuntu:~/code$ ll test
-rwxrwxr-x  1 abilatte abilatte    0 Jul 12 09:42 test*

abilatte@ubuntu:~/code$ chmod u-rw test 
abilatte@ubuntu:~/code$ ll test
---xrwxr-x  1 abilatte abilatte    0 Jul 12 09:42 test*
```



### chown (change owner)

>   chown - 修改文件拥有者和所属组
>

```shell
abilatte@ubuntu:~/code$ ll test
---xrwxr-x  1 abilatte abilatte    0 Jul 12 09:42 test*
# 修改文件所有者
abilatte@ubuntu:~/code$ sudo chown root test
[sudo: authenticate] 密码：          
abilatte@ubuntu:~/code$ ll test
---xrwxr-x  1 root     abilatte    0 Jul 12 09:42 test*
# 同时修改文件所有者和所属组
abilatte@ubuntu:~/code$ sudo chown root:root test
abilatte@ubuntu:~/code$ ll test
---xrwxr-x  1 root     root        0 Jul 12 09:42 test*
```



### chgrp (change group)

>   chgrp - 修改文件所属组

```shell
abilatte@ubuntu:~/code$ ll test
---xrwxr-x  1 root     root        0 Jul 12 09:42 test*

abilatte@ubuntu:~/code$ sudo chgrp abilatte test
abilatte@ubuntu:~/code$ ll test
---xrwxr-x  1 root     abilatte    0 Jul 12 09:42 test*
```
