## Vim模式

Vim中有四种模式, 分别是**普通模式**, **插入模式**, **命令行模式**, **可视模式**打开Vim时, 默认进入的是**普通模式**

在**普通模式**中可以输入`a` , `i`, `o`......进入**插入模式**, 输入`:`进入**命令行模式**, 输入`v` / `ctrl + v`进入**可视模式**. 在其他模式按ESC回到**普通模式**

|    命令    |              含义              |
| :--------: | :----------------------------: |
|    `i`     |    进入插入模式, 光标前插入    |
|    `a`     |    进入插入模式, 光标后插入    |
|    `o`     | 进入插入模式, 下面另起一行插入 |
|    `v`     |   进入可视模式, 字符选择模式   |
| `ctrl + v` |     进入可视模式, 竖选模式     |

## 普通模式

### 技巧相关

1.   Vim中命令是可以组合的
     1.   一种为`action + motion` , 将动作和方向组合
     2.   另一种为`num + action`, 表示动作重复的次数
2.   如果某个命令是有方向性的, 并且它用相同字母的大小写来代表不同的方向, 那么规律一般为 : 
     1.   小写字母代表向右, 向后, 向下
     2.   大写字母代表向左, 向前, 向上
     3.   一个著名的反例为`W` / `B`, 是按照大单词(以空格为分隔符)进行移动, 而不是向反方向移动



### 移动光标指令

|          命令           |                         含义                         |
| :---------------------: | :--------------------------------------------------: |
|           `h`           |                   向左移动一个字符                   |
|           `j`           |                   向下移动一个字符                   |
|           `k`           |                   向上移动一个字符                   |
|           `l`           |                   向右移动一个字符                   |
|     `:[n]` / `[n]G`     |                    移动到第`n`行                     |
|     `[n]-` / `[n]+`     |                 向上 / 向下移动`n`行                 |
|       `gg` / `G`        |                移动到第一 / 最后一行                 |
|           `^`           |             移动到本行的第一个非空白字符             |
|        `0` / `$`        |                   移动到行首 / 尾                    |
|        `b` / `w`        |                上 / 下一个单词的词首                 |
| `t {char}` / `T {char}` | 在当前行向右 / 左查找, 移动到`char`的前 / 后一个字符 |
| `f {char}` / `F {char}` |         在当前行向右 / 左查找, 移动到`char`          |



### 删除(`delete`)文本指令

Vim中的删除类似 Windows 中的剪切, 会将删除的内容复制到 VIm的"剪切板"中

|       命令        |         含义         |
| :---------------: | :------------------: |
|        `x`        |     删除一个字符     |
|       `dd`        |      删除当前行      |
| `[n]dd` / `d[n]d` |      删除`n`行       |
|   `:[m], [n]d`    |    删除`[m, n]`行    |
|    `d^` / `d$`    |  删除到行首 / 行尾   |
|    `dw` / `db`    | 删除后 / 前一个单词  |
|       `diw`       | 删除当前光标所在单词 |



### 撤销(`undo`)和重做指令

|    命令    | 含义 |
| :--------: | :--: |
|    `u`     | 撤销 |
| `ctrl + r` | 重做 |



### 复制(`yank`)和粘贴(`paste`)文本指令

|       命令        |        含义         |
| :---------------: | :-----------------: |
|       `yy`        |     复制当前行      |
| `[n]yy` / `y[n]y` |      复制`n`行      |
|   `:[m], [n]y`    |   复制`[m, n]`行    |
|    `yw` / `yb`    | 复制后 / 前一个单词 |
|     `p` / `P`     |  粘贴到下 / 上一行  |



### 修改文本指令

| 命令  |                     含义                     |
| :---: | :------------------------------------------: |
| `ci"` | 当光标在`""`中时, 删除引号内容并进入插入模式 |
| `ciw` |       删除光标所在单词, 并进入插入模式       |



### 查找文本指令

|     命令     |             含义              |
| :----------: | :---------------------------: |
| `/{pattern}` |  向后查找匹配`pattern`的内容  |
| `?{pattern}` |  向前查找匹配`pattern`的内容  |
|  `n` / `N`   | 按当前查找方向继续 / 反向查找 |



### 其他常用指令

|  命令  |     含义     |
| :----: | :----------: |
| `gg=G` | 修正代码缩进 |



## 命令行模式

| 命令  |          含义          |
| :---: | :--------------------: |
| `:w`  |        保存文件        |
| `:q`  | 在没有未保存修改时退出 |
| `:q!` | 丢弃未保存的修改并退出 |
| `:wq` |       保存并退出       |
