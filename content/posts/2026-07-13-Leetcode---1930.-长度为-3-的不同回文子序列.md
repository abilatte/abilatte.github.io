## 题目链接

>   [1930. 长度为 3 的不同回文子序列 - 力扣（LeetCode）](https://leetcode.cn/problems/unique-length-3-palindromic-subsequences/description/)



## 题目描述

>   给你一个字符串 `s` ，返回 `s` 中 **长度为 3** 的**不同回文子序列** 的个数。
>
>   即便存在多种方法来构建相同的子序列，但相同的子序列只计数一次。
>
>   **回文** 是正着读和反着读一样的字符串。
>
>   **子序列** 是由原字符串删除其中部分字符（也可以不删除）且不改变剩余字符之间相对顺序形成的一个新字符串。
>
>   -   例如，`"ace"` 是 `"abcde"` 的一个子序列。



## 思路

>   跟 [Leetcode - 3583. 统计特殊三元组](https://abilatte.github.io/posts/2026-07-13-leetcode---3583.-统计特殊三元组/) 思路差不多
>
>   但是这里求的是`不同的子序列`个数
>
>   长度为3的回文子序列一定形如 `side + mid + side`
>
>   枚举中间的`mid`, 再枚举两侧字符`side`
>
>   `pre_cnt[side] > 0`说明左侧出现过`side`
>
>   `suf_cnt[side] > 0`说明右侧出现过`side`
>
>   又因为题目统计的是不同的回文子序列
>
>   所以需要使用`counted[mid][side]`记录已经出现过的子序列



## 题解

```cpp
class Solution {
public:
    int countPalindromicSubsequence(string s) {
        // 前缀
        int pre_cnt[26]{};
        // 后缀
        int suf_cnt[26]{};
        // counted[a][b] 判断是否有 b a b 这个子序列
        bool counted[26][26]{};
        int ans = 0;
        // 先将所有字符记录
        for (char c : s) {
            suf_cnt[c - 'a']++;
        }
        
        // 枚举中间的字符
        for (char mid : s) {
            int mid_index = mid - 'a';
            // 将当前字符从右侧移除
            // 此时 pre_cnt 统计当前位置左侧，suf_cnt 统计当前位置右侧。
            // pre_cnt [0, mid - 1], suf_cnt [mid + 1, size - 1]
            suf_cnt[mid_index]--;
            // 从a到z 枚举两侧的字符
            for (int side = 0; side < 26; side++) {
                // 如果两侧都有, 且没有被记录
                if (pre_cnt[side] > 0 && suf_cnt[side] > 0 && !counted[mid_index][side]) {
                    // 答案+1
                    ans++;
                    // 记录当前组合
                    counted[mid_index][side] = true;
                }
            }
            // 将当前字符记录到前缀
            pre_cnt[mid_index]++;
        }

        return ans;
    }
};
```
