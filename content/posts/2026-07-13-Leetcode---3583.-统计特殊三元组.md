## 题目链接

>   [3583. 统计特殊三元组 - 力扣（LeetCode）](https://leetcode.cn/problems/count-special-triplets/)



## 题目描述

>   给你一个整数数组 `nums`。
>
>   **特殊三元组** 定义为满足以下条件的下标三元组 `(i, j, k)`：
>
>   -   `0 <= i < j < k < n`，其中 `n = nums.length`
>-   `nums[i] == nums[j] * 2`
>   -   `nums[k] == nums[j] * 2`
>   
>   返回数组中 **特殊三元组** 的总数。
>
>   由于答案可能非常大，请返回结果对` 10^9 + 7 `取余数后的值。
>

 

## 思路

>   枚举数组的中间下标`j`
>
>   找出数组中满足 `nums[i] == 2 * nums[j] == nums[k]` 的组合数量
>
>   可以枚举中间元素`j`, 找左右两边`2 * nums[j]`的数量, 将两者相乘并累加就是答案
>



## 题解

```cpp
class Solution {
public:
    int specialTriplets(vector<int>& nums) {
        // 题目要求取模
        const int MOD = 1'000'000'007;
        long long ans = 0;
        unordered_map<int, int> suf, pre;
        // 初始化后缀集合
        for (int x : nums) {
            suf[x]++;
        }
        // 枚举 j, x = nums[j]
        for (int x : nums) {
            // suf 范围 [j + 1, n - 1]
            suf[x]--;
            // 到这一步 pre 的范围为 [0, j - 1]
            ans += 1LL * suf[2 * x] * pre[2 * x];
            // [0, j]
            pre[x]++;
        }
        return ans % MOD;
    }
};
```
