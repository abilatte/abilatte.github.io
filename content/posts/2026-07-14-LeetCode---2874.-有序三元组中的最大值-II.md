## 题目链接

>   [2874. 有序三元组中的最大值 II - 力扣（LeetCode）](https://leetcode.cn/problems/maximum-value-of-an-ordered-triplet-ii/description/)



## 题目描述

>   给你一个下标从 **0** 开始的整数数组 `nums` 。
>
>   请你从所有满足 `i < j < k` 的下标三元组 `(i, j, k)` 中，找出并返回下标三元组的最大值。如果所有满足条件的三元组的值都是负数，则返回 `0` 。
>
>   **下标三元组** `(i, j, k)` 的值等于 `(nums[i] - nums[j]) * nums[k]` 。



## 思路

>   枚举中间元素, 维护左右侧最大值



## 题解

```cpp
class Solution {
public:
    long long maximumTripletValue(vector<int>& nums) {
        int n = nums.size();
        long long ans = 0;
        
        // 初始化前缀最大值为第一个元素
        int pre_max = nums[0];
        // 后缀最大值, suf_max[i] 保存的是 [i, n - 1] 区间的最大值 
        vector<int> suf_max(n);
        // 初始化 suf_max[n - 1] 的最后一个元素为 nums[n - 1]
        suf_max[n - 1] = nums[n - 1];
        
        // 从后往前初始化后缀容器, 保存各个区间最大值
        for (int i = n - 2; i >=0; i--) {
            suf_max[i] = max(suf_max[i + 1], nums[i]);
        }
        
        // 从前往后枚举中间元素
        for (int j = 1; j < n - 1; j++) {
            ans = max(ans, 1LL * (pre_max - nums[j]) * suf_max[j + 1]);
            // 维护前缀最大值
            pre_max = max(pre_max, nums[j]);
        }
        
        return ans;
    }
};
```
