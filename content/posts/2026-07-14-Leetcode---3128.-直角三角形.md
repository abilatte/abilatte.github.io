## 题目链接

>   [3128. 直角三角形 - 力扣（LeetCode）](https://leetcode.cn/problems/right-triangles/description/)



## 题目描述

>   给你一个二维 boolean 矩阵 `grid` 。
>
>   如果 `grid` 的 3 个元素的集合中，一个元素与另一个元素在 **同一行**，并且与第三个元素在 **同一列**，则该集合是一个 **直角三角形**。3 个元素 **不必** 彼此相邻。
>
>   请你返回使用 `grid` 中的 3 个元素可以构建的 **直角三角形** 数目，且满足 3 个元素值 **都** 为 1 。



## 思路

>   有三个顶点, 枚举直角顶点, 将每个直角顶点的直角边上 1 的数量相乘并累加即是答案
>
>   如果第`i`行有`row_sum`个 1 , `j`列有`col_sum`个 1 , 那么点`(i, j)`的直角三角形有`(row_sum - 1) * (col_sum - 1)`



## 题解

```cpp
class Solution {
public:
    long long numberOfRightTriangles(vector<vector<int>>& grid) {
        // 列数
        int cols = grid[0].size();
        // 每一列 1 的个数
        vector<int> col_sum(cols, 0);
        long long ans = 0;
        
        // 将每列 1 的个数填入容器
        for (const auto& row : grid) {
            for (int j = 0; j < cols; j++) {
                if (row[j] == 1) {
                    col_sum[j]++;
                }
            }
        }
        
        for (const auto& row : grid) {
            // 获取当前行 1 的数量
            int row_sum = reduce(row.begin(), row.end());
            for (int j = 0; j < cols; j++) {
                // 当前格子值为 1 时，可以作为直角顶点
                if (row[j] == 1) {
                    // 每行每列需要去掉当前的直角顶点
                    ans += 1LL * (col_sum[j] - 1) * (row_sum - 1);
                }
            }
        }
        return ans;
    }
};
```
