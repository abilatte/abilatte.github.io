## 题目链接

>   [3427. 变长子数组求和 - 力扣（LeetCode）](https://leetcode.cn/problems/sum-of-variable-length-subarrays/description/)



## 题目描述

>   给你一个长度为 `n` 的整数数组 `nums` 。对于 **每个** 下标 `i`（`0 <= i < n`），定义对应的子数组 `nums[start ... i]`（`start = max(0, i - nums[i])`）。
>
>   返回为数组中每个下标定义的子数组中所有元素的总和。
>
>   **子数组** 是数组中的一个连续、**非空** 的元素序列。



## 思路

>   对每一个`i`, 定义一个子数组`nums[start ... i](start = max(0, i - nums[i]))`
>
>   即每一个`i`, 取`[max(0, i - nums[i]), i]`的和



## 题解

```cpp
 class NumArray {
     // 前缀和数组
     vector<int> pre_sum;
 public:
     NumArray(vector<int>& nums) {
         // 设置数组大小为 nums.size() + 1
         pre_sum.resize(nums.size()  + 1);
         for (int i = 0; i < nums.size(); i++) {
             // pre_sum[i + 1] 表示 nums[0...i] 的和
             pre_sum[i + 1] = pre_sum[i] + nums[i];
         }
     }
     
     int sumRange(int left, int right) {
         // [0, right] - [0, left - 1]
         return pre_sum[right + 1] - pre_sum[left];
     }
 };
```
