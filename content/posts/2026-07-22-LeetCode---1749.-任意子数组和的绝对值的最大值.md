## 题目描述

>   给你一个整数数组 `nums` 。一个子数组 `[numsl, numsl+1, ..., numsr-1, numsr]` 的 **和的绝对值** 为 `abs(numsl + numsl+1 + ... + numsr-1 + numsr)` 。
>
>   请你找出 `nums` 中 **和的绝对值** 最大的任意子数组（**可能为空**），并返回该 **最大值** 。
>
>   `abs(x)` 定义如下：
>
>   -   如果 `x` 是负整数，那么 `abs(x) = -x` 。
>   -   如果 `x` 是非负整数，那么 `abs(x) = x` 。



## 思路

>   同 [LeetCode - 303. 区域和检索 - 数组不可变]({{< ref "posts/2026-07-20-LeetCode---303.-区域和检索---数组不可变.md" >}})



## 题解

```cpp
class Solution {
public:
    int maxAbsoluteSum(vector<int>& nums) {
        // 存放当前元素和, 当前元素和最大值, 当前元素和最小值
        int sum{}, mx{}, mn{};
        for (int& x : nums) {
            sum += x;
            mx = max(mx, sum);
            mn = min(mn, sum);
        }
        // 因为 mn 初始化为0, 所以mn <= 0
        return mx - mn;
    }
};
```
