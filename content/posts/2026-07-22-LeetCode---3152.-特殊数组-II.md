## 题目描述

>   如果数组的每一对相邻元素都是两个奇偶性不同的数字，则该数组被认为是一个 **特殊数组** 。
>
>   你有一个整数数组 `nums` 和一个二维整数矩阵 `queries`，对于 `queries[i] = [fromi, toi]`，请你帮助你检查 子数组 `nums[fromi..toi]` 是不是一个 **特殊数组** 。
>
>   返回布尔数组 `answer`，如果 `nums[fromi..toi]` 是特殊数组，则 `answer[i]` 为 `true` ，否则，`answer[i]` 为 `false` 。

## 题解

```cpp
class Solution {
public:
    vector<bool> isArraySpecial(vector<int>& nums, vector<vector<int>>& queries) {
        // 存放 [0, i] 是否是特殊数组
        vector<int> pre(nums.size());
        vector<bool> ans(queries.size());
        pre[0] = 0;
        // 初始化, pre[i]是否是特殊数组, 是由pre[i - 1]以及当前元素与前一元素的奇偶关系决定
        for (int i = 1; i < nums.size(); i++) {
            pre[i] = pre[i - 1] + (nums[i] % 2 == nums[i - 1] % 2);
        }
        // query[0] = left, query[1] = right, [left, right]是否是特殊数组, 看pre[right] - pre[left] == 0
        // 即 pre[left] == pre[right]
        for (int i = 0; i < queries.size(); i++) {
            auto& query = queries[i];
            ans[i] = ((pre[query[1]] == pre[query[0]]));
        }
        return ans;
    }
};
```
